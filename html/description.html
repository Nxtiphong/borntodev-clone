Description

คำสั่งเพิ่มเติมที่ไม่มีในชั้นเรียน

Tag : @keyframes

Example : 

element{
    animation: words-class 18s infinite;
}

@keyframes words{
    0%,20%{
        content: "HTML & CSS";
    }
    21%,40%{
        content: "JavaScript";
    }
    41%,60%{
        content: "Web Design";
    }
    61%,80%{
        content: " Debugging";
    }
    81%,100%{
        content: " Version Control";
    }
}

Tag keyframes นั้นใช้ในการกำหนดระยะเวลาในการทำงานโดยแบ่งเป็น % ต่อการทำงาน 1 ครั้ง และใช้ควบคู่กับ Tag animation ในการกำหนดระยะเวลา ชื่อของkeyframes และจำนวนรอบของการทำงาน

Tag : transition

Example : 

element{
    transition: all 0.25s ease;
}

คือคุณสมบัติที่ช่วยให้คุณสามารถกำหนดการเปลี่ยนแปลงในคุณสมบัติต่าง ๆ ของelement จากส่วนหนึ่งไปยังอีกส่วนหนึ่งโดยสามารถควบคุมความเร็วและลักษณะการเปลี่ยนแปลงนั้นได้

Tag : linear-gradient

Example : 

element{
    background: linear-gradient(to right,#ffc000,#ff2323);
}

linear-gradient ใน CSS คือคุณสมบัติที่ใช้สร้างพื้นหลังที่มีการเปลี่ยนแปลงสีเป็นลำดับตามเส้นทางเฉพาะที่กำหนด ช่วยให้สามารถสร้างพื้นหลังที่มีการผสมสีแบบเปลี่ยนแปลงได้ตามที่ต้องการ
