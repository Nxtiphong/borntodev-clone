Description

คำสั่งเพิ่มเติมที่ไม่มีในชั้นเรียน

Tag : ::before และ ::after

Example :

ตัวอย่างการเลื่อนไปมา Test

*ถ้าใช้คู่กับ @keyframes words ก็จะเปลี่ยนประโยคได้

.h3-topic span::before{
    content: "";
    color: black;
    animation: words 18s infinite;
}
.h3-topic span::after{
    content:"";
    position: absolute;
    width: calc(100% + 8px);
    height: 100%;
    background-color: #fff;
    right: -5px;
    animation: typing 18s infinite;
}

::before เป็น CSS pseudo-element ที่ใช้ในการสร้างเนื้อหาที่จะเพิ่มไปยังองค์ประกอบที่ถูกเลือก โดยไม่ต้องแก้ไข HTML ต้นฉบับ ส่วน ::after ใช้เหมือนกันแต่จะเปลี่ยนเป็นอยู่ข้างหลัง ตามcodeที่ผมใช้นั้น::beforeใช้ในการเปลี่ยนตัวอักษร และ::afterใช้ในการกำหนดให้มีสีขาวมาทับตัวอักษรให้ดูเหมือนถูกเลื่อนหายไป

Tag : transform

Example : 

element{
    transform: translateY(-10px);
}

เป็นคุณสมบัติใน CSS ที่ใช้ในการทำการแปลง (transform) องค์ประกอบที่ถูกเลือก, เช่น การเปลี่ยนตำแหน่ง, การหมุน, การขยาย, หรือการเอียง

Tag : appearance

Example : 

element{
    appearance: none;
}

ใช้กับTag select ทำให้ผลลัพธ์ที่แสดงผลไม่ได้รับการเข้ารหัสการแสดงผลจากบราวเซอร์และจะต้องกำหนดสไตล์เองเพื่อปรับแต่งลักษณะตามที่เราต้องการ

Tag : @keyframes typing

Example : 

@keyframes typing {
    10%,15%,30%,35%,50%,55%,70%,75%,90%,95% {
        width: 0%;
    }
    5%,20%,25%,40%,45%,60%,65%,80%,85%{
        width: calc(100% + 8px);
    }
}

เป็นการกำหนดkeyframes ที่จะปรับขนาดข้อความให้เป็น 0 พร้อมๆกับสร้างแถบสีขาวมาทับข้อความทำให้เหมือนข้อความถูกพิมพ์
